## Context

当前深度报告可输出宏观主题，但商品（黄金/白银/原油）不是强制段落，导致日报中偶发缺失或覆盖不足。用户需要稳定、客观、可验收的商品解读区块。

## Goals / Non-Goals

**Goals**
- 保证深度日报中稳定体现黄金/白银/原油三段解读
- 每段具备一致结构：事件/驱动/市场影响/定价状态/前瞻/可信度
- 最小改动，不新增 provider 和复杂解析模块

**Non-Goals**
- 不重构新闻抓取层
- 不新增数据库和缓存结构
- 不改现有调度策略

## Decisions

1. **Prompt 强约束优先**
   - 在 deep prompt 中直接要求输出固定三段商品模块，减少改动面

2. **轻量上下文增强**
   - 在 `core_deep.py` 从现有新闻摘要中抽取商品相关行，作为补充输入，提升 LLM 命中率

3. **失败可降级**
   - 保持原有 LLM 失败回退，不阻塞日报生成

## Risks / Trade-offs

- 风险：新闻文本质量不稳定导致商品段深度波动
- 应对：prompt 强制在数据不足时标注“数据不足+低可信度”，避免幻觉式结论
